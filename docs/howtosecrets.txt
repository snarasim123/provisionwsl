To create github secrets, encrypt and run the scripts,
.copy secrets-template.yaml to secrets.yaml. fill out the values you want to encrypt.
The keys are already set in the secrets-template.yaml file.
Fill in just the values.

.encrypt secrets.yaml file with ansible-vault
    ansible-vault encrypt secrets.yaml
Enter new password when prompted and verify secrets.yaml is encrypted.
save the password in plain text,  to a file called secrets.pass

.add the secrets.pass file to the project root. 

.do not checkin/commit secrets.yaml and secrets.pass files.

.run kickoff.ps1

To view the encrypted file contents,
    ansible-vault view secrets.yaml
Enter the password from secrets.pass when prompted.
